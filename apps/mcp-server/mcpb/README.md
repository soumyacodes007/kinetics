Kinetics Claude Desktop bundle assets live in `kinetics-claude-desktop/`.

The packed extension artifact is generated as:

- `apps/mcp-server/mcpb/kinetics-claude-desktop.mcpb`

The bundle prompts the installer for:

- `0G Private Key` as a sensitive value
- optional `0G RPC URL`
- optional `0G Storage Indexer URL`
- optional `Client Label`
